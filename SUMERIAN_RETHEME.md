# Sumerian Retheme Plan

This document tracks the proposed terminology changes for transitioning the project from a Greek to a Sumerian aesthetic.

## Vision
A Mesopotamian city-builder set in the Fertile Crescent. Visuals shift to mud-bricks, lapis lazuli, gold, and vibrant reed-mat patterns.

## Terminology Mapping

### Roles & Titles
| Greek/Generic | Sumerian Equivalent | Description |
|---|---|---|
| **Archon** | **En** | Lord/Priest-King of the city. |
| **Thetes** / Worker | **Mushkenu** | Commoner/Citizen-Worker. |
| **Hoplite** | **Shukur** | Spearman. |
| **Toxotes** | **Panu** | Archer. |
| **Machimos** | **Tukultu** | Clubman. |
| **Hippeis** | **Chariot** | Elite mounted units. |
| **Oikos** | **É** | The "House" or Estate (e.g., *É-Gal* for palace). |

### Constructs (Structures)
| Greek/Generic | Sumerian Equivalent | Description |
|---|---|---|
| **Agora** | **Mahiru** | Marketplace. |
| **Oracle** | **Ziggurat** | Temple-tower / House of God. |
| **Polis** | **Iru** | The City/Settlement. |
| **Barracks** | **Military Grounds** | Training area for warriors. |

### Name Examples
- **Male**: Gilgamesh, Ur-Nammu, Gudea, Enmerkar, Lugalbanda.
- **Female**: Enheduanna, Shala, Ninhursag, Puabi, Kubaba.

## Visual Palette
- **Mud-brick**: Earthy tans and browns.
- **Lapis**: Deep blues for elite structures.
- **Gold**: Highlighting the Ziggurat and En's residence.
- **Vegetation**: Date palms, reeds, and lush irrigated gardens.
